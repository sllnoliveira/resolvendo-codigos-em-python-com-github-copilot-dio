# Resolvendo Códigos em Python com IA

Este repositório contém a resolução de desafios propostos em aula utilizando Python.

O objetivo é praticar:

- Manipulação de strings
- Operações matemáticas
- Estruturas condicionais
- Entrada de dados
- Lógica de programação

Os exercícios foram desenvolvidos com apoio de ferramenta de IA (GitHub Copilot ou ChatGPT), utilizando as sugestões como apoio ao aprendizado.

## Lista de Exercícios

1. Concatenando Dados
2. Repetindo Textos
3. Operações Matemáticas Simples
4. Verificando Números Pares e Ímpares
5. Calculando Média de Notas
6. Verificando Palíndromos
